Select HTML elements with JavaScript

13 January 2021

There are more than one way to use javascript to select html elements and change its property.

  1. Using query selector method, this method can select element, a class or an id and this is the best method to use as it cover all types of elements-
    document.querySelector()
  2. Get elements by tag name-
    document.getElementsByTagName()
  3. This method will give an array of selected elements because it takes class an input-
    document.getElementsByClassName()
  4. This method gives only one item not an array because it takes id as input-
    document.getElementById()
